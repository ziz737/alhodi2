## 📁 هيكل الملفات

```
alhoudi-real-estate/
├── index.html          ← الصفحة الرئيسية
├── property.html       ← صفحة تفاصيل العقار
├── requests.html       ← صفحة الطلبات العقارية
├── admin.html          ← لوحة التحكم (الإدارة)
├── save.php            ← ملف معالجة البيانات والصور
├── .htaccess           ← إعدادات الخادم
├── data/
│   ├── properties.json ← بيانات العروض العقارية
│   └── requests.json   ← بيانات الطلبات العقارية
├── uploads/            ← مجلد رفع الصور (يُنشأ تلقائياً)
└── assets/
    ├── css/
    │   └── style.css   ← ملف التصميم الرئيسي
    └── images/
        └── logo.png    ← شعار الموقع
```

---

## 🚀 كيفية رفع الملفات للاستضافة

1. ارفع **جميع الملفات والمجلدات** إلى مجلد `public_html` أو `www` في استضافتك
2. تأكد من دعم الخادم للـ **PHP 7.4+**
3. تأكد من صلاحيات الكتابة على مجلد `data` و `uploads`
4. افتح المتصفح وادخل رابط موقعك

### صلاحيات المجلدات
```bash
chmod 755 uploads/
chmod 755 data/
chmod 644 data/properties.json
chmod 644 data/requests.json
```

## ✨ الميزات

### الصفحة الرئيسية
- عرض العروض العقارية بطاقات جميلة
- بحث وفرز حسب (نوع العقار - المدينة)
- زر الطلبات العقارية
- تصميم متجاوب مع الجوال

### صفحة تفاصيل العقار
- عنوان بارز وواضح
- بيانات مرئية (المدينة - المساحة - التاريخ - السعر)
- عرض جميع الصور
- زرا التواصل (واتساب + مكالمة)

### صفحة الطلبات العقارية
- عرض جميع الطلبات بطاقات
- بيانات الطلب (النوع - المدينة - الميزانية)
- زر التواصل واتساب

### لوحة التحكم (admin.html)
- إحصائيات شاملة
- إدارة العروض العقارية (إضافة / تعديل / حذف)
- رفع صور متعددة
- إدارة الطلبات العقارية (إضافة / تعديل / حذف)
